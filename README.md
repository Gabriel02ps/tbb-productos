## ðŸ˜Š How to run

Before starting, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed on your machine.

```sh
- Clone this repo:
$ git clone https://github.com/Gabriel02ps/tbb-productos.git

- Enter directory:
$ cd tbb-productos

- Install dependencies:
$ npm install

- Launch the app:
$ npm run dev
```
